Analysis and Workflow

In this document I combine the analysis of the Penguin dataset LINK, witch is part of the palmerpenguins package, with focus on my R programming workflow, supported bu Yan Holtz’s course LINK.

Work habits

Restart R very often and re-run your under-development script from the top.

  • Restart R Ctrl+Shift+F10

  • Re-run all code up to the current line with Ctrl+Alt+B
